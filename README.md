## 📌 스터디 규칙

- 평일: 매일 1문제  
- 주말: 토·일 합해서 1문제  
- (면접, 경조사 등 제외)

## 📊 문제 난이도

- 양심껏 선택 (아래 기준 권장)
  - 백준: 실버 ~ 골드
  - 프로그래머스: 레벨 2 ~ 3

## 🍀 커밋 규칙

- 각자 이름으로 branch 생성 후 파일 commit 및 PR
- 커밋 메시지: `년월일_문제이름` (예: `251124_달팽이`)
- `main` branch에 merge

## 🧀 PR 규칙

- PR 제목: `년월일_이름` (예: `251124_지성현`)
- PR 내용: 제공된 템플릿 사용
